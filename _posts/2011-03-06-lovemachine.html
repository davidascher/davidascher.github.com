--- layout: post title: Love Machine summary: a service which makes it easy for sites to delegate recording and displaying of "liking" operations categories: [status:concept, difficulty:hard, target:RPs] ---

Motivation

Public storage of something akin to a Facebook like, but authed by Persona and for public benefit.

Inspiration

Disqus, Facebook Like

Design Questions

The current version keeps track of which Persona ID declared their love for what URL. There are lots of open questions about what to do with that data, the degree to which it should be private and if so exactly how, etc. For now I'm taking the stance (which should be communicated to users) that this is a loud, public w00t.

Implementation notes

There are obvious scaling questions if this kind of thing were to be adopted by lots of sites. The current code is a trivial node server backed by redis.

I'm finding that the links don't always respond to clicks, which could be me somehow running into clickjacking protection, or otherwise missing some basic CSS something.

Status

I have a very very trivial implementation currently working, and powering the "Feedback" interaction with the megaphone icon at the bottom of each of the posts on this page. Source is available on github.
